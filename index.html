Мен сайтты қалай құрдым?

Мен осы сайтты жасау үшін мынадай қадамдарды орындадым:

  1. HTML файл құру: Алдымен мен HTML құжатын жасадым. HTML — бұл веб-беттерді құруға арналған негізгі тіл. Мен HTML құрылымын жасадым, мысалы, сайттың басты тақырыбын, мәзірін және негізгі бөлімдерін белгіледім. HTML арқылы менің сайттың құрылымы орнатылды, ал кейінірек оған CSS стилдерін қосып, әр элементтің орналасуын реттедім.
  2. CSS стилін қосу: Сайтты әдемі әрі пайдалануға ыңғайлы ету үшін мен CSS қолдандым. CSS арқылы сайттың фонын, шрифттерін, батырмаларын, тақырыптар мен мәтіндердің түсін өзгертуге мүмкіндік алдым. Мысалы, көгілдір фон, сары батырмалар мен көк мәтіндер таңдадым. Бұл менің сайтымды визуалды түрде тартымды етті.
  3. JavaScript қосу: Сайтқа динамикалық элементтер қосу үшін мен JavaScript пайдаланылды. Мен батырмаларды басқан кезде сайттың мазмұнын көрсету немесе жасыру үшін JavaScript кодын жаздым. Мысалы, "Портфолио" батырмасын басқан кезде портфолио бөлімі көрінеді, ал "WIX Сайтқа өту" батырмасын басқан кезде менің WIX сайтқа сілтеме ашылады.
  4. GitHub-қа жүктеу: Сайттың барлық кодын GitHub репозиторийіме жүктедім. GitHub — бұл менің сайтымның кодтарын сақтау және оларды басқалармен бөлісу үшін өте ыңғайлы құрал. Репозиторийде мен барлық HTML, CSS және JavaScript файлдарын сақтадым.
  5. GitHub Pages арқылы сайтты жариялау: GitHub репозиторийінде мен сайтты GitHub Pages арқылы жарияладым. GitHub Pages — бұл GitHub репозиторийінен автоматты түрде сайт жасау қызметі. Осы арқылы менің сайтым интернетте жарияланып, кез келген адам оны көре алады.
  6. WIX сайтыма сілтеме жасау: Мен сайтта WIX платформасында жасаған жеке сайтқа сілтеме жасадым. Бұл сілтеме арқылы менің басқа жобаларыммен танысуға болады. WIX — бұл қолданушыларға оңай сайт құруға мүмкіндік беретін құрал, мен оны өзімнің жеке портфолио жобаларым үшін қолданамын.
